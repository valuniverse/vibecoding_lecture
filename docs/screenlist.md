# Screen List - AI 웰니스 코치 (MVP)

## 화면 목록

| # | 화면명 | 설명 |
|---|---|---|
| S-01 | 스플래시 | 앱 로딩 화면 |
| S-02 | 온보딩 - 환영 | 첫 진입 환영 + 시작하기 버튼 |
| S-03 | 온보딩 - 프로필 설정 | 이름, 목표, 경험, 부상 부위 입력 |
| S-04 | 홈 | 오늘 체크인 상태, 최근 기록 요약 |
| S-05 | 데일리 체크인 | AI와 자연어 대화 (컨디션 파악) |
| S-06 | 추천 대시보드 | 오늘의 운동 루틴 + 식단 팁 + 추천 근거 |
| S-07 | 운동 수행 | 운동 체크리스트 + 타이머 |
| S-08 | 피드백 입력 | 수행 여부, 만족도, 코멘트 |
| S-09 | 히스토리 | 날짜별 운동 기록 목록 |
| S-10 | 히스토리 상세 | 특정 날짜의 운동 + 피드백 상세 |

---

## 화면 상세

### S-01 스플래시
- 앱 로고 + 로딩 인디케이터
- 자동 전환: 신규 유저 → S-02 / 기존 유저 → S-04

---

### S-02 온보딩 - 환영
- 서비스 소개 문구 (1~2줄)
- `시작하기` 버튼 → S-03

---

### S-03 온보딩 - 프로필 설정
**입력 항목**
- 이름 (닉네임) - 텍스트 입력
- 운동 목표 - 단일 선택
  - 체중 감량 / 근력 향상 / 체력 유지 / 스트레스 해소
- 운동 경험 - 단일 선택
  - 초급 / 중급 / 고급
- 주의 부위 - 다중 선택 (선택사항)
  - 어깨 / 허리 / 무릎 / 손목 / 없음

**액션**
- `저장하기` → S-04 (홈)

---

### S-04 홈
**구성 요소**
- 오늘 날짜 + 인사 메시지
- 체크인 상태 배너
  - 미완료: "오늘 체크인 아직 안 했어요" + `체크인 하기` 버튼 → S-05
  - 완료: 오늘 추천 요약 카드 → S-06
- 최근 7일 운동 완료 현황 (캘린더 또는 도트 표시)
- 히스토리 보기 버튼 → S-09

---

### S-05 데일리 체크인
**구성 요소**
- 채팅 UI (AI 메시지 + 유저 입력)
- 텍스트 입력창 + 전송 버튼
- 빠른 응답 버튼 (예: "피곤해요", "괜찮아요", "30분 가능")

**파악 항목**
- 수면 시간/질
- 피로도
- 근육통 부위
- 가용 시간
- 특이사항 (선택)

**액션**
- 체크인 완료 → S-06 (추천 대시보드) 자동 이동

---

### S-06 추천 대시보드
**구성 요소**
- 추천 근거 (1줄 텍스트)
- 오늘의 운동 루틴 카드
  - 종목명, 세트 × 횟수, 예상 총 소요 시간
- 오늘의 식단 팁 (1줄)
- `운동 시작` 버튼 → S-07
- `루틴 바꿔줘` 버튼 → AI 재추천 (1회 가능)

---

### S-07 운동 수행
**구성 요소**
- 운동 목록 (체크리스트)
  - 종목명, 세트/횟수
  - 각 항목 완료 체크
- 전체 경과 시간 타이머
- `운동 종료` 버튼 (중간 종료 가능)

**액션**
- 전체 완료 또는 종료 → S-08 (피드백)

---

### S-08 피드백 입력
**구성 요소**
- 수행 여부 선택 (3지선다)
  - 완료했어요 / 일부만 했어요 / 못 했어요
- 만족도 선택 (별점 1~5)
- 한줄 코멘트 입력 (선택, 텍스트)
- `저장하기` 버튼

**액션**
- 저장 → S-04 (홈) 이동 + 히스토리 누적

---

### S-09 히스토리
**구성 요소**
- 날짜별 운동 기록 리스트
  - 날짜, 운동 종목 요약, 만족도 별점
- 항목 탭 → S-10 (상세)

---

### S-10 히스토리 상세
**구성 요소**
- 해당 날짜의 체크인 요약 (컨디션 파악 결과)
- 수행한 운동 루틴 전체
- 피드백 내용 (수행 여부, 만족도, 코멘트)
