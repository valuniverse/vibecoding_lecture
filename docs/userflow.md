# User Flow - AI 웰니스 코치 (MVP)

## 전체 플로우 개요

```
앱 실행
    ↓
[로그인 화면]
    ↓ Google 로그인
[신규 유저 (프로필 없음)]           [기존 유저 (프로필 있음)]
    ↓                                        ↓
온보딩 (프로필 설정)                  앱 실행 (홈 화면)
    ↓                                        ↓
    └──────────────── 데일리 체크인 ──────────┘
                          ↓
                  추천 대시보드 확인
                          ↓
                      운동 수행
                          ↓
                   피드백 입력
                          ↓
                히스토리 누적 & 학습
```

---

## 1. 로그인 (모든 유저)

**진입:** 앱 최초 실행 또는 비로그인 상태 접근

1. 로그인 화면 표시
2. [Google로 시작하기] 버튼 클릭
3. Google OAuth 팝업 → 계정 선택 → 승인
4. Supabase Auth 세션 생성
5. 분기:
   - `profiles` 테이블에 프로필 있음 → 홈(`/`) 이동
   - 프로필 없음 (첫 로그인) → 온보딩(`/onboarding`) 이동

---

## 2. 온보딩 (신규 유저만, 첫 로그인 시)

**진입:** Google 로그인 후 프로필 미등록 상태

1. 기본 프로필 입력
   - 이름 (닉네임)
   - 운동 목표 (체중감량 / 근력향상 / 체력유지 / 스트레스 해소)
   - 운동 경험 (초급 / 중급 / 고급)
   - 부상 또는 주의 부위 (선택, 다중 선택)
2. 프로필 저장 (`profiles` INSERT, id = auth.uid()) → 홈 화면 이동

---

## 2. 데일리 체크인

**진입:** 앱 실행 시 자동 시작 (당일 체크인 미완료 상태)

### 신규 유저 (히스토리 없음)
```
AI: "안녕하세요! 오늘 컨디션 어때요?"
유저: 자유 응답
AI: "수면은 어느 정도 됐나요?"
유저: 응답
AI: "오늘 운동에 쓸 수 있는 시간은요?"
유저: 응답
AI: "혹시 오늘 특별히 피하고 싶은 부위나 운동 있어요?"
유저: 응답 (없으면 패스)
→ 체크인 완료 → 추천 대시보드 이동
```

### 기존 유저 (히스토리 있음)
```
AI: "어제 [하체 운동] 하셨는데, 오늘 다리 상태는 어때요?"
유저: 응답
AI: "오늘 가용 시간은요? (어제는 40분이었어요)"
유저: 응답
→ 맥락 파악 완료 시 추가 질문 생략
→ 체크인 완료 → 추천 대시보드 이동
```

**목표:** 3~5턴 이내 완료

---

## 3. 추천 대시보드

**진입:** 데일리 체크인 완료 후 자동 이동

### 화면 구성
```
┌─────────────────────────────────┐
│  오늘의 추천                     │
│  "수면 부족이라 강도를 낮췄어요" │  ← 추천 근거 1줄
├─────────────────────────────────┤
│  운동 루틴 (예상 35분)           │
│  ① 스쿼트  3세트 × 12회         │
│  ② 런지    3세트 × 10회         │
│  ③ 플랭크  3세트 × 30초         │
├─────────────────────────────────┤
│  오늘의 식단 팁                  │
│  "운동 전 바나나 1개 추천해요"   │
├─────────────────────────────────┤
│  [운동 시작]   [루틴 바꿔줘]     │
└─────────────────────────────────┘
```

### 분기
- **운동 시작** → 운동 수행 화면 (타이머/체크리스트)
- **루틴 바꿔줘** → AI에게 재추천 요청 (1회 가능)

---

## 4. 운동 수행

**진입:** 추천 대시보드 → 운동 시작 버튼

1. 운동 목록 체크리스트 형태로 표시
2. 각 종목 완료 시 체크
3. 전체 완료 또는 중간 종료 → 피드백 화면 이동

---

## 5. 피드백 입력

**진입:** 운동 수행 완료 또는 종료 후

```
1. 수행 여부 선택
   [완료했어요] [일부만 했어요] [못 했어요]

2. 만족도 선택 (1~5점)
   ★ ★ ★ ★ ★

3. 한줄 코멘트 (선택)
   "________________"

→ [저장하기]
```

저장 완료 → 홈 화면 이동 + 히스토리 누적

---

## 6. 히스토리 & 학습

- 피드백 저장 시 백그라운드에서 자동 처리
- 학습 데이터: 수행 여부, 만족도, 코멘트, 운동 종목
- 다음 체크인 시 AI 참조 → 질문 최소화 및 추천 개인화

---

## 화면 목록 요약

| 화면 | 진입 조건 |
|---|---|
| 로그인 | 비로그인 상태 접근 시 리다이렉트 |
| 온보딩 | 로그인 후 프로필 미등록 시 |
| 홈 | 로그인 + 프로필 등록 상태 |
| 데일리 체크인 (AI 대화) | 당일 체크인 미완료 |
| 추천 대시보드 | 체크인 완료 후 |
| 운동 수행 | 대시보드 → 운동 시작 |
| 피드백 입력 | 운동 완료/종료 후 |
| 히스토리 | 홈에서 접근 (MVP 선택 포함) |
