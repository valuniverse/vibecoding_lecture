# PRD - AI 웰니스 코치 (MVP)

## 1. 개요

매일 달라지는 컨디션을 자연어 대화로 파악하고, AI가 히스토리 기반 맞춤 운동 루틴과 식단을 코칭해주는 개인 맞춤형 웰니스 코치 서비스.

## 2. 목표

운동을 꾸준히 하고 싶지만 매일 무엇을 해야 할지 고민하는 직장인/홈트족/운동 초중급자가, 죄책감 없이 "오늘의 나"에 맞는 운동을 할 수 있게 한다.

MVP 성공 기준:
- 데일리 체크인 완료율 70% 이상 (7일 연속 사용 기준)
- 추천 운동 수행률 60% 이상

---

## 3. 핵심 기능 (MVP 범위)

### 3-1. 자연어 데일리 체크인
- 앱 진입 시 AI가 오늘 컨디션을 묻는 대화 시작
- 파악 항목: 수면 시간/질, 피로도, 근육통 부위, 가용 시간, 특이사항
- 이전 기록이 있으면 AI가 맥락을 참조해 질문 최소화 (예: "어제 하체 했는데 오늘도 괜찮아요?")
- 대화 3~5턴 이내 완료 목표

### 3-2. 맞춤 추천 대시보드
- 체크인 완료 후 오늘의 운동 루틴 + 식단 팁 제공
- 추천 근거를 1줄로 표시 (예: "수면 부족이라 강도를 낮췄어요")
- 운동 루틴: 종목명, 세트/횟수, 예상 소요 시간
- 식단 팁: 간단한 한 줄 조언 (MVP에서는 식단 상세 계획 제외)

### 3-3. 피드백 수집
- 운동 후 3가지만 수집: 수행 여부(완료/부분/스킵), 만족도(1~5), 한줄 코멘트(선택)
- 수집 데이터는 다음 추천에 반영

---

## 4. MVP 제외 항목

- 식단 상세 플랜 / 칼로리 계산
- 웨어러블 연동
- 소셜/커뮤니티 기능
- 영상 가이드
- 푸시 알림 커스터마이징

---

## 5. 유저 플로우 (핵심)

```
앱 실행
  → 데일리 체크인 (AI 대화, 3~5턴)
  → 오늘 추천 대시보드 확인
  → 운동 수행
  → 피드백 입력 (완료/만족도/코멘트)
  → 히스토리 누적 → 다음 날 추천에 반영
```

---

## 6. 기술 스택 (제안)

| 항목 | 선택 |
|---|---|
| AI 대화 | Claude API (claude-sonnet-4-6) |
| 프론트엔드 | Next.js |
| 백엔드/DB | Supabase |
| 배포 | Vercel |

---

## 7. 일정 (MVP)

| 주차 | 작업 |
|---|---|
| 1주 | 유저 프로필 + 체크인 대화 UI |
| 2주 | 추천 대시보드 + AI 연동 |
| 3주 | 피드백 수집 + 히스토리 저장 |
| 4주 | QA + 베타 테스트 |
